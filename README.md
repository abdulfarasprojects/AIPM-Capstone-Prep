# AIPM-Capstone-Prep
Demo Day Prep Frameworks – Question Banks, Templates, and Scorecards  This repo contains all the resources I used to prepare for my AI Product Management demo day, where my capstone project was selected as a runner‑up from the cohort.

## What’s inside:

Full judge question bank from past cohorts, grouped into 14 themes (problem, ICP, why AI, wedge, evaluation, GTM, pricing, etc.)

Answer structure template for Q&A (direct answer → evidence → tradeoff → next step)

4‑minute demo structure guide with timing marks and section goals

Mock judge scorecard (Notion + PDF) to self‑assess across key rubric categories

10‑item prep checklist to make sure your core answers are crisp before demo day

Example fills from my capstone (Amanah.Finance) so you can see how each framework looks in practice

## Who it’s for:

Product Faculty students and AI PMs

Founders and builders preparing for demo days

Educators and coaches who want reusable templates for their cohorts

## How to use this repo:

Start with the question bank to see what judges actually ask.

Draft your answers using the answer structure template.

Script your 4‑minute demo using the structure guide.

Score yourself with the mock judge scorecard to find weak spots.

Run through the 10‑item checklist the day before your demo.

Everything here is free to fork, remix, and adapt for your own demos.

If you improve a template or add new patterns, consider opening a PR so the next cohort can start one step ahead.
